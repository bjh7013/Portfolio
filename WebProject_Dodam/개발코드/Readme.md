# 프로젝트명 -도담도담

#### 개요
도담도담 - 유아 교육용 플렛폼
언텍트 시대에 부합하는 유아 교육용 플랫폼을 만드는 것을 목표로 함, 기존 교육 플랫폼과의 차별점으로 OCR과 TTS를 이용한 편의 사항과 클래스 기능을 이용하여 선생과 학생을 연결시키는 기능을 제공함.


#### **팀명 : 벙글이들**

팀장 : 백정훈

팀원 : 박지은, 이원우, 이용수, 진승화
